What is Messiah? We are planning to build a service to help save lives and prevent economic losses through mechanisms to predict, prevent, or manage the impact of natural disasters, as accurately as possible and feasible using the dataset procurable.
To start with, the user needs to enter the name of the city/country, he/she wants to enquire about, in the above search bar. After pressing Enter, the user will be guided to the various services we provide, in the Dashboard. The service would:
  • Provide real-time map(satellite imaging too) of the user's given location, where he/she can get a glimpse of the calamity.
  • Predict the magnitude of a natural disaster, in the given location, set by the user.
  • Provide the user with valuable precautions which can be implemented well in advance based on the geography and topography of the location the user resides in. These tips will be such as "build houses with x type of pillars to ensure sturdiness at times of earthquake".
  • Show data of previous natural disasters of the area, and grossly present the damage done by them to property and life.
  • API for developers to work further on the project
  • SMS services for the user to connect with his family members in times of distress.
